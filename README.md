# 큐브 이스케이프 수유점 - 입장 안내 가이드

## 프로젝트 개요
방탈출 카페 "큐브 이스케이프 수유점"의 입장 전 안내 가이드 웹페이지입니다.
아이패드에서 재생되며, 고객에게 필수 안내사항을 전달한 후 구글폼으로 연결됩니다.

---

## 작업 내용

### 1. 기본 구조 설정
- 기존 HTML/CSS/JS 파일 기반으로 작업
- 아이패드 전체 화면에 맞는 반응형 레이아웃 적용 (`100vw x 100vh`)

### 2. 콘텐츠 추가
#### 자물쇠 안내 섹션 추가
- **금고** (마녀의 꿈, 사라진 천사들 테마)
  - 키패드 UI (1-9, C, 0, E) 그래픽 구현
  - 비밀번호 입력 → E버튼 → 초록불 → 손잡이 오른쪽 돌려 열기
  - C버튼으로 초기화, 3회 초과 시 5분간 잠김

- **도어락** (더 큐브, 사라진 천사들 테마)
  - 키패드 UI (1-9, x, 0, #) 그래픽 구현
  - 비밀번호 입력 → #버튼 누르기
  - 횟수 제한 없음

#### 자물쇠 이미지 적용
- 방향 자물쇠, 숫자 자물쇠, 마스터 자물쇠, 4버튼 자물쇠
- 아이콘 → 실제 PNG 이미지로 교체

### 3. 폰트 변경
- 전체 폰트를 `J직지고딕-Regular.otf` → `JikjiGothic-Regular.otf`로 적용

### 4. UI/UX 개선

#### 스페이스바 일시정지 기능
- 스페이스바: 일시정지/재개 토글
- ← 화살표: 이전 화면
- → 화살표: 다음 화면
- ESC: 처음으로

#### 상단 섹션 탭 네비게이션
- 주의사항 / 힌트시스템 / 자물쇠 안내 3개 탭
- 현재 섹션 활성화 표시
- 탭 클릭 시 해당 섹션으로 이동

#### 버튼 스타일 개선
- 아웃라인 → 채워진 배경색 스타일로 변경
- 라운딩 적용 (`border-radius: 16px~25px`)
- 호버/터치 시 색상 변화 (기본 #333 → 호버 #555 → 터치 #666)
- 터치 반응형 추가 (`touch-action: manipulation`, `:active` 스타일)

#### 레이아웃 다듬기
- 힌트폰 UI, 금고/도어락 UI에 그림자 효과
- 전체적으로 둥근 모서리 적용

### 6. 직원용 자물쇠 화면 수정

## 사용 방법
1. 아이패드에서 웹페이지 열기
2. 전체 화면 모드로 실행
3. 방문 여부에 따라 O/X 버튼 선택
4. 가이드 시청 후 구글폼 작성

### 키보드 단축키 (테스트용)
- `Space`: 일시정지/재개
- `←`: 이전 화면
- `→`: 다음 화면
- `ESC`: 처음으로
